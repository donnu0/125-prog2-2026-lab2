# whoami

Команда whoami виводить ім’я поточного користувача.

## Обробка помилок

- Якщо передані зайві аргументи — команда виводить usage-повідомлення.
- Якщо виникає внутрішня помилка — команда повертає код 1.

## Коди завершення

- 0 — успішне виконання
- 1 — помилка виконання
- 2 — помилка аргументів
