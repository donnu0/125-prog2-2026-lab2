# cat command

## Що робить команда
Команда cat виводить вміст одного або кількох файлів у стандартний потік виводу (stdout).

## Обробка некоректних аргументів
- Якщо не передано жодного файлу → виводиться usage повідомлення і код 2
- Якщо файл не існує → помилка в stderr
- Якщо передано директорію → помилка
- Якщо немає прав доступу → помилка

## Можливі проблеми
- Відсутність файлу
- Відсутність прав доступу
- Помилка читання або запису

## Коди завершення
- 0 — успіх
- 1 — помилка виконання (файл не знайдено, доступ заборонено)
- 2 — неправильне використання

## Особливості
- Підтримка декількох файлів
- Не змінює стан shell
- Не використовує stdin
