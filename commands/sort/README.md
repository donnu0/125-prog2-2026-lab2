# Команда sort

**Що робить ваша команда?** Команда sort зчитує текст із файлу, сортує рядки за алфавітом і виводить їх на екран (у stdout).

**Які некоректні аргументи ви обробляєте?** Якщо користувач не вказав файл або вказав більше одного, команда виводить підказку "Usage: sort <filename>" (у stderr) і завершує роботу з кодом 2.

**Що може піти не так під час виконання?** Файлу може не існувати (FileNotFoundError) або до нього може не бути доступу (PermissionError). Код перехоплює ці помилки (принцип Zero Trust), виводить коротке повідомлення і не дає програмі "впасти".

**Які коди завершення ви повертаєте і чому?** 0 - все успішно.
1 - помилка виконання (немає файлу, немає доступу, інша системна помилка).
2 - помилка вводу (неправильна кількість аргументів).
